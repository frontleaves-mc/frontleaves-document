# 机械动力核心

机械动力（Create）是整合包的机械核心模组，提供从动力传输到自动化加工的完整系统。

## 鼓风机

鼓风机（Encased Fan）的处理速度极快，几乎可以做到即时处理。鼓风机可用于冶炼、洗涤、烟熏等操作。

## 蓝图炮

蓝图炮（Schematicannon）用于打印建筑蓝图。每份火药可发射 `150` 次，使用时需要合理规划火药资源。

## 批量加工

- **批量压制**：机械压机可以同时处理多个物品
- **批量锯切**：机械锯可以同时处理多个物品

## 传送带

传送带最大长度为 `32` 格，可以建造较长的输送线路而无需频繁中继。
