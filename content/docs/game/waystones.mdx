# 传送石

传送石（Waystones）提供传送点标记与传送功能。

## 生成与充能

- 野生传送石之间的区块距离为 `40`，比较稀疏
- 传送石充能时间为 `80` ticks（约 4 秒）

## 传送费用

| 传送类型 | 费用 |
|:---------|:-----|
| 同维度传送 | 经验 `0.03` |
| 跨维度传送 | `35` 级经验 |
| 传送板费用 | 正常费用的一半 |
| 冷却时间 | `600` ticks（30 秒） |

跨维度传送需要积累足够的经验等级，短距离传送费用较低。
