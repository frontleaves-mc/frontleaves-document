# 锋楪认证

锋楪认证是基于 Yggdrasil 协议的外置登录认证服务，用于管理 Minecraft 游戏账户与角色。

## 快速开始

如果你是群内玩家，推荐直接使用**群内提供的客户端**，已内置锋楪认证配置，开箱即用，无需手动设置。
